## 4. INTERACTIVE MICRO-INTERACTIONS & MICRO-ANIMATIONS (JS & CSS)

Integrate this exact vanilla JavaScript framework to drive micro-interactions across the UI:

### A. Magnetic Hover Pull (`data-magnetic`)
Make interactive actions follow the cursor slightly, creating a tactile pull effect on desktop devices:
```javascript
document.querySelectorAll("[data-magnetic]").forEach((el) => {
  let raf = null, tx = 0, ty = 0, cx = 0, cy = 0;
  const STRENGTH = 12;

  function loop() {
    tx += (cx - tx) * 0.18;
    ty += (cy - ty) * 0.18;
    el.style.transform = `translate(${tx.toFixed(2)}px, ${ty.toFixed(2)}px)`;
    if (Math.abs(cx - tx) > 0.05 || Math.abs(cy - ty) > 0.05) {
      raf = requestAnimationFrame(loop);
    } else {
      raf = null;
    }
  }

  el.addEventListener("mousemove", (e) => {
    const r = el.getBoundingClientRect();
    const dx = (e.clientX - (r.left + r.width / 2)) / (r.width / 2);
    const dy = (e.clientY - (r.top + r.height / 2)) / (r.height / 2);
    cx = dx * STRENGTH;
    cy = dy * STRENGTH;
    if (!raf) raf = requestAnimationFrame(loop);
  });

  el.addEventListener("mouseleave", () => {
    cx = 0; cy = 0;
    if (!raf) raf = requestAnimationFrame(loop);
  });
});
```

### B. Apple-style Scroll Reveals
Elements fade and translate dynamically as they enter the viewport using an Intersection Observer:
- Styles:
  ```css
  .apple-reveal { opacity: 0; transform: translateY(28px); will-change: transform, opacity; transition: opacity 0.9s cubic-bezier(0.2, 0.7, 0.2, 1), transform 0.9s cubic-bezier(0.2, 0.7, 0.2, 1); }
  .apple-scale { opacity: 0; transform: scale(0.96); will-change: transform, opacity; transition: opacity 0.9s cubic-bezier(0.2, 0.7, 0.2, 1), transform 0.9s cubic-bezier(0.2, 0.7, 0.2, 1); }
  .is-in.apple-reveal { opacity: 1; transform: translateY(0); }
  .is-in.apple-scale { opacity: 1; transform: scale(1); }
  ```
- Observer script:
  ```javascript
  const revealObserver = new IntersectionObserver((entries) => {
    entries.forEach((entry) => {
      if (entry.isIntersecting) {
        entry.target.classList.add("is-in");
        revealObserver.unobserve(entry.target);
      }
    });
  }, { threshold: 0.12, rootMargin: "0px 0px -6% 0px" });

  document.querySelectorAll(".apple-reveal, .apple-scale").forEach(el => revealObserver.observe(el));
  ```

### C. Animated Numerical Counters
Increment statistics smoothly when they scroll into view:
- HTML usage: `<div class="stat-number" data-counter="1250">0</div>`
- Script:
  ```javascript
  const easeOutCubic = (t) => 1 - Math.pow(1 - t, 3);
  const counterObserver = new IntersectionObserver((entries) => {
    entries.forEach((entry) => {
      if (!entry.isIntersecting) return;
      counterObserver.unobserve(entry.target);
      const el = entry.target;
      const target = parseInt(el.getAttribute("data-counter"), 10);
      const duration = 1600;
      const startTime = performance.now();

      function tick(now) {
        const elapsed = Math.min(1, (now - startTime) / duration);
        el.textContent = Math.round(easeOutCubic(elapsed) * target).toLocaleString();
        if (elapsed < 1) requestAnimationFrame(tick);
        else el.textContent = target.toLocaleString();
      }
      requestAnimationFrame(tick);
    });
  }, { threshold: 0.4 });

  document.querySelectorAll("[data-counter]").forEach(el => counterObserver.observe(el));
  ```

### D. Animated Card Grid Staggering
Delay grid items progressively so they cascade into place:
```javascript
const gridObserver = new IntersectionObserver((entries) => {
  entries.forEach((entry) => {
    if (entry.isIntersecting) {
      const items = entry.target.querySelectorAll(".glass-card, .stat-card");

}, { threshold: 0.1 });

document.querySelectorAll(".grid-container").forEach(el => gridObserver.observe(el));
```

## 6. DEVELOPMENT RULES

1. **Accessibility**: All buttons must have descriptive `aria-label` tags. Use `aria-expanded` and `aria-haspopup` for interactive elements.
2. **SEO**: Set clean heading structures (`h1` -> `h2` -> `h3`), custom title tag blocks, and meta description blocks.
3. **Reduced Motion**: Respect system preferences by removing transform animations and disabling looped orbits when user prefers reduced motion:
   ```css
   @media (prefers-reduced-motion: reduce) {
     .apple-reveal, .apple-scale { opacity: 1; transform: none; transition: none; }
     .orbit, .glow, .glow-text { animation: none !important; }
   }
   ```
4. **Interactive Responsiveness**: The hamburger menu toggles a `.menu-open` class that transitions mobile navigation items from a hidden tray into view using elegant animations.
